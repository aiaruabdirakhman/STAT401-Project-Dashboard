# STAT401-Project-Dashboard

Project Overview

This project was developed by Aiaru Abdirakhman, Hadiya Khan, and Nuray Taghiyeva. It applies big data analytics, Apache Spark, and Python-based text mining to study parliamentary activity in Türkiye related to earthquakes, housing policy, urban transformation, and fiscal measures. Using a large corpus of TBMM laws, written questions, and parliamentary research motions, the project analyzes how legislative and oversight activity evolved between 1999 and 2026, with a focus on the last five years.

An interactive dashboard presents the findings, highlighting key trends in earthquake-related legislation, housing and reconstruction policies, and parliamentary oversight patterns.
